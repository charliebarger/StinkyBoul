StinkyBoul Privacy Policy

Last updated: April 8, 2026

StinkyBoul is a Chrome extension that helps users save hunt codes in a chosen order and autofill tag details from the browser side panel.

What StinkyBoul stores

  • Saved access token for extension activation
  • A browser-specific identifier used to validate the activated token
  • Saved hunt codes and their display order

How data is used

This data is used only to activate the extension, remember a user's hunt codes, and autofill tag details when the user runs the extension.

Where data is stored

Hunt codes and activation state are stored locally in Chrome extension storage. Activation requests are checked against a Supabase backend that stores access tokens and the browser identifier that claimed each token.

What StinkyBoul does not do

  • It does not sell user data
  • It does not use data for advertising
  • It does not track browsing history for analytics
  • It does not share personal data with third parties except as needed to run the activation backend

Permissions

StinkyBoul requests Chrome permissions needed to open the side panel, store data locally, identify the active tab, and inject autofill logic into the current page when the user explicitly runs the program.
