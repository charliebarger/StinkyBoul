# StinkyBoul Chrome Web Store Draft

## Name

StinkyBoul

## Short Description

Save hunt codes in priority order and autofill tag details from a Chrome side panel.

## Detailed Description

StinkyBoul helps you keep your hunt codes organized and run them in priority order from a simple side panel.

Use StinkyBoul to:

- save hunt codes so they are still there when you come back later
- drag and reorder codes into the exact priority you want
- run codes from top to bottom until one succeeds
- autofill tag details into the active page from Chrome

StinkyBoul is built for a focused single-purpose workflow: saving hunt codes and autofilling tag details when you explicitly start the run.

## Single Purpose

StinkyBoul lets users save hunt codes and autofill tag details from a side panel into the current browser tab.
