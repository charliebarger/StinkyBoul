Install StinkyBoul

Download the current Chrome extension zip below, then load it into Chrome as an unpacked extension.

Download StinkyBoul ZIP

How to install

  1. Download the zip file above.
  2. Unzip it on your computer.
  3. Open chrome://extensions in Chrome.
  4. Turn on Developer mode.
  5. Click Load unpacked.
  6. Select the unzipped extension folder.

How to use

  1. Open the StinkyBoul side panel in Chrome.
  2. Enter your access code.
  3. Add or reorder hunt codes.
  4. Open the CPW page and run the program.
